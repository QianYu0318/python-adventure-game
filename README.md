# python-adventure-game
# 🧙 Python 冒險者小遊戲

## 📖 專案介紹
這是一個使用 Python 製作的簡單回合制戰鬥遊戲。
玩家需要擊敗怪物才能獲勝。

---

## 🎮 遊戲玩法
- 玩家初始血量：100
- 怪物血量：80
- 每回合可選擇：
  - 1️⃣ 攻擊
  - 2️⃣ 防禦
- 打敗怪物即可獲勝 🎉

---

## 🚀 如何執行程式

請在 VS Code 終端機輸入：

```bash
python game.py
```
